1. .width()/.height()
- 브라우저의 높이값을 알아낸 뒤, 4개의 콘텐츠 박스의 높이에 적용합니다.

2. 이벤트 연결하기
- 화면을 스크롤할 때마다 모션을 실행할 것이므로 스크롤 이벤트를 연결합니다.

3. .scrollTop()
- 현재 스크롤된 거리 만큼을 저장하거나 실제 해당 문서를 스크롤시킵니다.

4. 조건문(if문)
- 전체 문서를 조건문으로 4개의 구역으로 나눈 뒤, 해당 역역에 스크롤이 위치했을 시 버튼을 활성화합니다.

5. 반복문(for문)
- 반복되는 코드를 반복문을 이용하여 간결하게 줄입니다.

6. .css()
- 마우스의 움직임에 따라 서브 이미지의 css 속성인 가로, 세로 좌표 값을 계속해서 변경합니다.

7. mousewheel.js
- 마우스 휠의 동작에 따라 실행되는 모션을 연결합니다.

8. .offset().top()
- 문서의 제일 산단을 기준으로 4개의 영역의 세로 위치값을 알아냅니다.
